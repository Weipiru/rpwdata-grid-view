# RPWDataGridView

#### 介绍
一个超级简洁的表格控件，支持固定多列，目前支持基本数据加载，点击事件

![Preview](./res/p4.gif)

#### 使用说明

自行打包依赖

1. `RPWDataGridView<T>` 表格控件 T为数据类型
1. 主要接口
 + `build(...)` 添加列
 + `setRowBuildListener`  行数据构建接口，在这设置每列显示的数据
 + `setDataSource` 设置数据源
